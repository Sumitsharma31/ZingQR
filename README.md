# ZingQR - Free QR Code Generator

![ZingQR Logo](https://zingqr.app/og-image.png)

**ZingQR** is a premium, lightning-fast QR code generator designed for simplicity and professional results. Create, customize, and download high-quality QR codes in seconds—absolutely free and with no signup required.

## ✨ Features

- 🚀 **Instant Generation**: Create QR codes in real-time as you type.
- 🎨 **Deep Customization**: Adjust colors, shapes, and styles to match your brand.
- 📥 **High-Quality Exports**: Download your codes in PNG or SVG formats.
- 🛡️ **Privacy First**: No account required. Your data remains yours.
- 📱 **Mobile Responsive**: Fully optimized for all devices.
- ⚡ **Premium UI**: Featuring smooth animations, glassmorphism, and a sleek dark-themed loader.

## 🛠️ Tech Stack

- **Frontend**: [React](https://reactjs.org/) + [Vite](https://vitejs.dev/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **QR Engine**: [qr-code-styling](https://github.com/kozakdenys/qr-code-styling)

## 🚀 Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v18 or higher recommended)
- npm or yarn

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Sumitsharma31/ZingQR.git
   ```

2. **Navigate to the project directory:**
   ```bash
   cd ZingQR
   ```

3. **Install dependencies:**
   ```bash
   npm install
   ```

4. **Start the development server:**
   ```bash
   npm run dev
   ```

5. **Build for production:**
   ```bash
   npm run build
   ```
